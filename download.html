Тарифи після пробного періоду

Basic

Для невеликої майстерні.

  • Ремонти
  • Клієнти
  • Документи
  • Базові QR

Pro

Для активного сервісного центру.

  • Все з Basic
  • Склад
  • Фінанси
  • Звіти
  • База знань

Cloud

Для онлайн-роботи з клієнтами.

  • Все з Pro
  • Кабінет клієнта
  • Синхронізація
  • Захищені QR

Enterprise

Для мережі майстерень.

  • Все з Cloud
  • Кілька філій
  • Ролі доступу
  • API

Як встановити

  1. Натисніть “Завантажити”.
  2. Запустіть EXE-файл.
  3. При першому запуску введіть дані майстерні.
  4. Отримайте Workshop ID та починайте роботу.
