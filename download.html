Завантажити для Windows

Beta-версія для тестування першими майстрами.

⬇ Завантажити ONLINE Service Manager Windows 10 / 11 · Beta · v0.2.12

Як встановити

1. Завантажити

Натисніть кнопку та збережіть EXE-файл на компʼютер.

2. Запустити

Якщо Windows покаже попередження, натисніть “Докладніше” → “Виконати”.

3. Зареєструвати майстерню

При першому запуску введіть назву майстерні, телефон, адресу та email.

Онлайн-поновлення

Програма може перевіряти нові версії через сайт ONLINE Service Manager. Коли вийде новий реліз, користувач зможе завантажити оновлення прямо з програми.

  • Перевірка останньої версії через update_manifest.json
  • Завантаження EXE з GitHub Release
  • Підготовка до автоматичного встановлення
