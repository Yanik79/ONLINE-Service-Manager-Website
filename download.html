Завантажити для Windows

Файл буде доступний після завантаження тестової збірки.

⬇ Завантажити Beta Windows 10 / 11 · Beta-версія · Для тестування

Як встановити

1. Завантажити

Натисніть кнопку завантаження та збережіть EXE-файл на компʼютер.

2. Запустити

Windows може попередити, що програма нова. Натисніть “Докладніше” → “Виконати”.

3. Тестувати

Створіть кілька ремонтів, перевірте документи, QR-код, базу знань і зручність роботи.

Що перевірити тестувальникам

Нам важливо зрозуміти, наскільки програма зручна для реальної майстерні.

  • Чи зручно створювати ремонт?
  • Чи зрозумілі статуси ремонту?
  • Чи коректно працюють документи?
  • Чи корисна База знань?
  • Що потрібно додати в першу чергу?
