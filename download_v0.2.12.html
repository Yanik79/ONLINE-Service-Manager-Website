Завантажити для Windows

Beta-версія для тестування першими майстрами.

⬇ Завантажити ONLINE Service Manager Windows 10 / 11 · Beta · v0.2.12

Як встановити

1. Завантажити

Натисніть кнопку та збережіть EXE-файл на компʼютер.

2. Запустити

Якщо Windows покаже попередження, натисніть “Докладніше” → “Виконати”.

3. Зареєструвати майстерню

При першому запуску введіть назву майстерні, телефон та адресу.

Тарифи

Beta-версія дозволяє протестувати програму перед запуском платних тарифів.

  • Basic — базовий облік ремонтів і клієнтів.
  • Pro — склад, фінанси, звіти, розширені можливості.
  • Cloud — кабінет клієнта, синхронізація, захищені QR.
  • Enterprise — кілька майстерень, ролі, API.
